# Анализ качества данных (Data Quality Analysis)

Проект для проверки качества данных и влияния предобработки на обучение модели логистической регрессии с использованием Jupyter Notebook.

## Содержание
- [Описание проекта](#описание-проекта)
- [Состав репозитория](#состав-репозитория)


## Описание проекта
Данный проект является учебным. Проект направлен на изучение проблемы качества данных и влияния предобработки на качество обучения модели 
логистической регрессии.

## Состав репозитория
- `DataQual.ipynb` — Jupyter Notebook с кодом анализа.
- `README.md` — документация.
- `train.csv` — датасет для анализа.
