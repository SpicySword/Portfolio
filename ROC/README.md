# Оценка точности модели, переобучение, регуляризация

Проект для оценки точности обучения модели с использованием Jupyter Notebook.

## Содержание
- [Описание проекта](#описание-проекта)
- [Состав репозитория](#состав-репозитория)


## Описание проекта
Данный проект является учебным. Проект направлен на закрепления знаний о математическом смысле метрик TPR, FPR и изучение построения ROC-кривой, графика Precision-Recall.

## Состав репозитория
- `ROC.ipynb` — Jupyter Notebook с кодом.
- `README.md` — документация.
- `athletes.csv` — датасет для обучения моделей.
